- [ ] Menu closes after clicking a link
- [ ] Menu closes when clicking backdrop
- [ ] Menu closes when pressing Escape key
- [ ] Header has solid white background (no transparency)
- [ ] No tiny rectangle boxes visible near header

### Homepage
- [ ] Hero section displays correctly with purple gradient

- [ ] Animations are smooth (no jank)
- [ ] Scrolling is smooth
- [ ] No console errors
